# 작업 로그 — about-me Q&A 봇 만들기

날짜: 2026-06-26

## 요청 → 작업 요약

### 1. 요청: "새 에이전트 만들기"
- 작업: 기존 에이전트 목록 확인(k-drama-expert, quick-recipe-creator, single-react-dev, single-server-specialist) 후, 어떤 에이전트인지 상세 질문.

### 2. 요청: "about-me.md 내용만 근거로 답하는 Q&A 봇. 파일에 없으면 '몰라요.'"
- 작업: `.claude/agents/about-me-qna.md` 에이전트 생성.
  - 유일한 정보 출처: `week-3/quest-3/storyme/about-me.md`
  - 답변 전 반드시 파일을 Read
  - 파일에 없는 내용은 추론·창작 없이 정확히 **"몰라요."**
  - 외부 지식/검색 금지
  - 모델: haiku (단순 조회용)

### 3. 요청: "오상원 직업이 뭐야?"
- 답변: **"회사원입니다. 물류회사에서 일하고 있습니다."** (파일 근거 ✅)

### 4. 요청: "좋아하는 음식이 뭐야?"
- 답변: **"몰라요."** (파일에 없음 → 규칙대로 동작 ✅)

### 5. 요청: "취미에 대해 알려줘"
- 답변: **"오상원의 취미는 자기개발과 독서입니다."** (파일 근거 ✅)

### 6. 요청: "대화로그 요약해서 같은 폴더에 저장"
- 작업: 이 파일(`worklog.md`)을 `storyme` 폴더에 저장.

## 결과물
- `.claude/agents/about-me-qna.md` — about-me Q&A 에이전트 (신규)
- `week-3/quest-3/storyme/worklog.md` — 이 작업 로그 (신규)

## 검증
- 파일에 있는 정보(직업·취미)는 정확히 답하고, 없는 정보(좋아하는 음식)는 "몰라요."로 응답 → 의도대로 동작 확인.
