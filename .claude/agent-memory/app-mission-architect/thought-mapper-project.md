**treeview:** 사용자가 정의한 앱(이름 확정, 2026-07-18). 두서없는 긴 글·문서(PDF·기사)를 넣으면 핵심 요약 + 마인드맵으로 정리해주는 개인용 생각정리 도구. 우선 본인용, 완성도 확보 후 무료+부분유료로 공개 배포 목표.

- MISSION.md 위치: `C:\afm-3-weekend\treeview\MISSION.md` (폴더명도 treeview로 확정. 이전 가칭 ThoughtMapper/생각지도, 이전 폴더명 thought-mapper 는 폐기).
- v1(무료): 입력→요약(연관성 범위 a: 넣은 내용 안에서만)→마인드맵(요약+맵 기본/트리형/그물망 선택/아웃라인 토글)→PNG·MD 내보내기 + **모바일 최적화(1급 요구사항)**.
- **모바일 우선(mobile-first)·반응형·터치 친화**가 v1 핵심 요구사항. 이유: 스마트폰 사용자 비중이 높다고 사용자가 명시.
- v2(유료): 새 관점·빠진 고리 제안(b), 과거 메모와 연결(c), (후보)웹검색 아이디어 확장.
- 유료 경계: 기능은 다 무료로 열되 **월 생성 횟수 제한**, 무제한+심화기능이 유료. 광고 안 붙임.
- UI: 단일 입력창 중심 미니멀, 여백 기반 구분, 맵 목록은 화면 크기에 따라 적응(데스크톱 사이드바/모바일 하단시트).

**Why:** 사용자가 "생각이 많고 산만해서" 정리를 앱에 위임하고 싶어 함. 정리 부담 자체를 없애는 게 핵심 가치. 배포 대상이 폰 사용자라 모바일 최적화가 필수.
**How to apply:** 다음 단계는 dev-kickstart(개발 구조·DEV.md). 기능 추가 논의 시 "정리 부담 최소화 / 미니멀·모바일 우선 UI" 원칙과 무료-유료 경계(월 횟수 제한)를 기준으로 스코프 판단.
