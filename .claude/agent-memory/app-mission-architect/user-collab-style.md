사용자(리테일 트레이더, BTC 비트겟 선물 / NQ HFM MT5 CFD 매매)의 협업 선호:

- **판단 위임을 편하게 함**: 확신이 없는 결정(예: v1을 어느 시장부터 시작할지)은 "데이터 모으기·빠른 안정화 쉬운 쪽으로 알아서 정해달라"며 근거 있는 추천에 맡긴다. → 애매한 항목은 근거를 붙여 강하게 추천하고 진행하는 편이 잘 맞는다.
- **전사(transcription)보다 교정·교육을 원함**: 자기 규칙(손절·익절·필터)을 정확히 못 짚을 때 "가이드 요청"을 명시한다. 위험한 값(예: 1회 매매에 시드 10% 리스크)은 이유와 함께 교정해주는 것을 환영한다.
- 재량매매를 명시적 규칙으로 바꾸려는 동기가 강함(진입근거 명확화 + 심리 안정). 관련 킵 아이디어: 전략·리스크·복기 멀티에이전트 + 매매일지 실패패턴 분석.

**How to apply**: 미션/스펙 정의 시 빈칸을 그냥 되묻기보다, 합리적 기본값을 근거와 함께 제시하고 "확인 필요 가정"으로 표시해 사용자가 검토·수정하게 하는 흐름이 효과적이었다.
