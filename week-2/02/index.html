HTML / CSS / JavaScript (의미만 살린 예제)

HTML은 구조, CSS는 꾸미기, JavaScript는 "변화(움직임/상태)"를 만들어요.

HTML

화면에 "무엇이 있는지" 정해요. (뼈대)

예: 버튼/박스/텍스트를 배치

<div id="box">Hello</div>

CSS (Tailwind)

"어떻게 보일지" 정해요. (색/크기/모양)

CSS로 만든 “멈춰있는” 모습 (JS 없이 그대로 고정)

🙂

포인트: 위 보라 박스/크기/둥근 모서리/그림자는 전부 Tailwind 클래스(CSS)로만 만든 거예요. 

<div class="flex h-28 w-28 items-center justify-center rounded-2xl bg-purple-600 text-5xl shadow-lg">
  🙂
</div>

JavaScript

클릭하면 클래스/텍스트를 바꿔서 "변화"를 만들어요.

상태: OFF
🙂

포인트: JS는 "style.transform"을 계속 바꿔서 움직이는 것처럼 보여줘요.
