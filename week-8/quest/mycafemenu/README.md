# ☾ 으스스 카페 메뉴판 — 제출물

유령·할로윈 컨셉 카페 **‘으스스’** 메뉴판 디자인 결과물입니다.
디자인은 **Gemini 2.5 Flash Image**로 세련된 배경 비주얼을 생성하고, 정확한 한글 메뉴명·가격은 **PIL로 정밀 합성**하는 하이브리드 방식으로 제작했습니다. (이미지 생성 모델은 한글을 정확히 렌더링하지 못하므로, 배경은 AI가·글자는 코드가 담당)

## 📦 제출물 체크리스트

| 요구사항 | 파일 |
|---|---|
| ✅ 메뉴판 이미지 (PNG) | `으스스_메뉴판_v1.png` (프레임형·추천), `으스스_메뉴판_v2.png` (무드형) |
| ✅ 메뉴판 이미지 (PDF) | `으스스_메뉴판_v1.pdf`, `으스스_메뉴판_v2.pdf` |
| ✅ 카페 컨셉 한 단락 설명 | `CONCEPT.md` |
| ✅ 에이전트 대화 스크린샷 | `대화_스크린샷.png` (실제 터미널 창 캡쳐 — 요구사항 논의 화면) |

## 🎨 디자인 스펙
- 컬러: 검정 + 주황 (할로윈 상징색)
- 사이즈: 1728 × 2368 px (3:4 세로, A4 인쇄 대응 / 200 DPI PDF)
- 폰트: BlackHanSans(타이틀·섹션·강조), 맑은고딕(메뉴·설명·가격)
- 두 가지 배경 버전
  - **v1 프레임형**: 우아한 주황 라인 프레임 + 보름달 + 유령 + 하단 잭오랜턴 줄 → 가독성·정돈감 우수 (기본 추천)
  - **v2 무드형**: 시네마틱 글로우 보름달 + 묘지 실루엣 + 안개 → 분위기 극대화

## 🛠 재현 방법
```bash
# 1) Gemini로 배경 생성 (텍스트 없는 비주얼 2종)
python gen_bg.py all

# 2) 배경 위에 메뉴 텍스트 정밀 합성 → 최종 PNG + PDF
python overlay_menu.py v1     # 프레임형
python overlay_menu.py v2     # 무드형
```

## 📁 파일 구성
```
mycafemenu/
├─ 으스스_메뉴판_v1.png / .pdf      # 최종 메뉴판 (프레임형·추천)
├─ 으스스_메뉴판_v2.png / .pdf      # 최종 메뉴판 (무드형)
├─ CONCEPT.md                       # 카페 컨셉 한 단락 설명
├─ 대화_스크린샷.png                # 에이전트 요구사항 대화 캡쳐
├─ gen_bg.py                        # Gemini 배경 생성 스크립트
├─ overlay_menu.py                  # 메뉴 텍스트 합성 스크립트
├─ bg_v1_raw.png / bg_v2_raw.png    # Gemini 원본 배경 (소스)
├─ fonts/BlackHanSans-Regular.ttf   # 타이틀 폰트
└─ .gemini_key.txt                  # API 키 (gitignore)
```
